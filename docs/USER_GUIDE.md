'''
# Polygame 用户指南

欢迎使用 Polygame！本指南将帮助您快速上手。

## 1. 平台简介

Polygame 是一个纯娱乐性质的模拟预测市场交易平台。您可以使用平台赠送的虚拟积分，对体育、电竞、娱乐等各类事件进行预测交易，体验预测市场的乐趣。

**重要声明**：本平台不涉及任何真实货币交易，所有积分均为虚拟，不可提现。

## 2. 快速上手

### 2.1 注册账户

1.  访问 Polygame 网站或打开移动 App。
2.  点击“注册”（Sign Up）。
3.  输入您的用户名、邮箱和密码。
4.  注册成功后，您将自动获得 **10,000** 虚拟积分作为启动资金。

### 2.2 浏览市场

-   在“市场”（Markets）页面，您可以浏览所有可交易的事件。
-   您可以使用分类、状态或搜索功能来查找您感兴趣的市场。

### 2.3 进行交易

1.  **选择市场**：点击您感兴趣的市场进入详情页面。
2.  **选择结果**：每个市场都有多个可能的结果选项（例如，A队获胜 vs B队获胜）。
3.  **买入（Buy）**：如果您认为某个结果会发生，您可以“买入”该结果的份额。份额的价格（0.01-0.99之间）代表了市场认为该结果发生的概率。
4.  **卖出（Sell）**：如果您持有某个结果的份额，并且想在市场结算前锁定利润或减少损失，您可以“卖出”您的份额。

### 2.4 市场结算

-   当事件结果确定后，市场将被“结算”（Resolved）。
-   如果您持有的份额是正确的结果，您将为每份份额获得 1.00 虚拟积分。
-   如果您持有的份额是错误的结果，该份额将归零。

## 3. 核心概念

### 份额价格 (Share Price)

-   份额的价格在 0.01 到 0.99 虚拟积分之间浮动。
-   价格直接反映了市场对该结果发生可能性的实时预测。例如，价格为 0.75 意味着市场认为该结果有 75% 的可能性发生。

### 虚拟积分 (Virtual Credits)

-   平台内唯一的交易货币。
-   用于购买份额，并在卖出或结算时获得。

### 投资组合 (Portfolio)

-   在“投资组合”页面，您可以查看您当前的虚拟积分余额、所有持仓以及最近的交易历史。

## 4. 常见问题 (FAQ)

**Q: 我可以充值真实货币吗？**

A: 不可以。Polygame 是一个纯粹的模拟平台，不支持任何真实货币的充值或提现。

**Q: 虚拟积分用完了怎么办？**

A: 目前平台仅在注册时赠送初始积分。我们未来可能会增加每日签到等活动来补充积分。

**Q: 交易是实时的吗？**

A: 是的，份额的价格会根据所有用户的买卖行为实时变动。

**Q: 我可以创建自己的市场吗？**

A: 目前只有管理员可以创建市场，以确保市场的质量和准确性。
